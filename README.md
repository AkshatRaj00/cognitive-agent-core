🚀 Overview

Mythos AI Core is a futuristic autonomous cognitive agent framework designed for:

🧠 Multi-Model AI Reasoning
⚡ Autonomous Runtime Execution
📊 Real-Time Telemetry Monitoring
🔐 Defensive Security Analysis
🌐 AI Tool Orchestration
💾 Persistent Memory Systems
🤖 Intelligent Decision Pipelines

This project combines AI orchestration, telemetry systems, memory architecture, runtime execution engines, and cyber-defense concepts into one modular framework.

🛠 Installation
Clone Repository
git clone https://github.com/AkshatRaj00/cognitive-agent-core.git
cd cognitive-agent-core
Install Requirements
pip install -r requirements.txt
🔑 Environment Variables

Create a .env file:

GROQ_API_KEY=your_api_key_here

streamlit run app_dashboard.py
